These are the code files for a Capstone class project. All work contained in thie project is the product of Nathaniel Alkire, Matthew Arthur, and Bradley Thompson.

If downloaded, this project will not run. Only the code files are contained within, not the dependencies or other necessary aspects due to the constraints of uploading this 
project to GitHub.

This project was a demonstration in response to an RFP from the class instructor. The goal of this project was to create a schedule planner for a student to use while signing up 
for classes in order to see what classes were available and what their schedule may look like.

This project uses C#, .Net Core, Microsoft Azure, and MySql.
